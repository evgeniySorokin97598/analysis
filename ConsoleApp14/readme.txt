Описание:
Данное приложение позволяет анализировать html страницу с целью подсчёта одинаковых слов в тексте 


Инструкция:

для записи результатов в БД в файле app.config укажите строку подключения


при запуске программы :
1.Введите url адрес нужного сайта.
2.Введите название для html файла, который будет сохранён на жёсткий диск.
3.после вывода статистики введите название таблицы, в которую нужно сохранить результат работы. Таблица будет создана автоматически.

логи с ошибками хранятся в папке с программой  bin\Debug\netcoreapp3.1\Logs
скаченные документы в папке bin\Debug\netcoreapp3.1\HTML files
